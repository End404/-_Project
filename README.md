计算机程序设计艺术：程序设计（programming），是给出解决特定问题程序的过程，软件开发过程中的重要步骤。

程序设计往往以某种程序设计语言为工具，给出这种语言下的程序。

程序设计过程一般包括分析、设计、编码、测试、调试等不同阶段。
